Tôi cần làm trang web cho: Mạng xã hội thu nhỏ dành cho học tập.
Về database thì tôi đã thiết kế để ở trong file Cơ sở dữ liệu_StudyNet.pdf.
Hệ thống của tôi có phân quyền: USER hoặc ADMIN. Trước tiên bạn cứ phát triển các trang cho sinh viên đã, sau đó mới phát triển cho tôi dashboard dành cho admin. User với admin sau khi đăng nhập vào thì sẽ được chuyển đến trang khác nhau. Admin thì đến trang dashboard của admin còn user thì đến trang newfeed của sinh viên.

Hiện tại tôi đã thiết kế xong frontend của HomePage, Login, Register nhưng chưa kết nối với backend. Vậy nên tôi cần bạn kết nối với backend cho tôi. Viết các API cơ bản trước tiên: API đăng ký, đăng nhập.

Bây giờ tôi cần bạn thiết kế tiếp frontend cho tôi, theo concept mà tôi đang làm. Trước tiên hãy thiết kế trang Newfeed ( dành cho user: sinh viên) gần giống như là newfeed của Facebook. 
- Ở sidebar có: 
    + Nhóm của bạn (tức là những nhóm mà sinh viên đã tham gia)
    + Lọc theo môn (liệt kê sẵn ở dưới các môn học: Vd như: Toán học, Vật lý, Hóa học, Lập trình ...) để khi nhấn vào môn học bất kỳ thì chỉ có các bài viết có tag là môn học đó sẽ hiện lên. 

- Ở navbar có: 
    + Bảng tin ( tức là newfeed tôi mô tả bên trên)
    + Nhóm học tập ( tức là toàn bộ các nhóm của hệ thống)
    + Có thanh tìm kiếm ( để tìm kiếm bài viết, tài liệu, nhóm...)
    + Chuông ( Thông báo )
    + Avatar + Tên của user ( Khi ấn vào dấu v bên cạnh tên user sẽ hiện Trang cá nhân, Cài đặt, Đăng xuất)
